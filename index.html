Modernizing Our Volunteer App: A Secure and Scalable Cloud Solution

Transitioning to Oracle Cloud Infrastructure (OCI)

January 17, 2025

Why Modernize Our Infrastructure?

  • Current System Challenges: Issues with Expo/Firebase setup
  • Goal: Enhanced Control and Flexibility: More control over our infrastructure.
  • Goal: Improved Performance and Scalability: System that can handle increased load.
  • Goal: Self-Hosting Advantage: Self-hosting provides more control and cost efficiency.

What is Oracle Cloud Infrastructure?

  • Cloud Computing Overview: Basic explanation of cloud computing.
  • OCI Overview: Cloud platform by Oracle for resources.
  • Key Benefits of OCI: Scalability, security, cost-effectiveness, control.
  • Relevance to Our Project: OCI will host and manage our application.

System Architecture Overview

High-level components: React Native App, Backend API, Database, Real-time Communication.

The Foundation of our Network: Virtual Cloud Network (VCN)

A private data center in the cloud.

  • Isolation: Resources isolated in the VCN.
  • Control: Control network traffic.
  • Security: Foundation for secure communication.

Inside the VCN: Subnets and Security

  • Subnet Segregation: Separate resources for security.
  • Security Rules: Security Lists and NSGs to control traffic.
  • Principle of Least Privilege: Restricting access.

Key Resources in OCI

  • Compute Instances: VM images for our servers.
  • Load Balancer: Manages incoming traffic.
  • Database (managed or compute): Persistent storage.
  • Object Storage: Static resources storage.
  • Websocket Server: For real-time communication.
  • NAT Gateway: For private resources to access the internet.

Development vs. Production Environments

Development

  • Local development with docker.
  • Staging for pre-production testing.

Production

  • Complete infrastructure setup.
  • Designed for high availability and traffic.

CI/CD manages code changes.

Managing Data and Location Updates in Real-Time

  • Database Choice: (e.g., PostgreSQL, MySQL). Data migration from Firebase.
  • Real-time Solution: WebSockets for real-time communication.
  • Data Storage: Location and volunteer cleanup information storage.
  • Communication: WebSockets push updates to connected apps.

Security and Reliability are Key

  • Security Measures: Encryption, secure access, firewalls, updates.
  • Backups and Recovery: Data backup strategy.
  • High Availability: The load balancer for fault tolerance.
  • Monitoring: OCI monitoring tools for insights and alerting.

Technology Stack

  • React Native: Front-end framework.
  • Node.js/Python: Backend framework.
  • PostgreSQL/MySQL: Database choice
  • WebSockets: Real-time data communication.
  • Docker/Kubernetes: (optional) Containerization and orchestration

Expected Benefits

  • Scalability: Handle more users.
  • Control: Full control of infrastructure.
  • Performance: App is faster and more reliable.
  • Cost-Effective: Optimize costs on resource usage.
  • Enhanced Security: Secure system.
